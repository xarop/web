## UX Engineer

[Que s ***UX Engineer*** segons Google?](https://uxe.withgoogle.com/)

*Web UX/UI Designer & Front End Developer* -> **Desenvolupador web i dissenyador d'interfícies amb àmplia experiència professional en desenvolupaments web i multimèdia.**

Com a enginyer UX, el meu treball combina els principis de l'experiència d'usuari amb tasques de desenvolupament i programació:

-   **Planificar i dissenyar pàgines web i aplicacions**: Utilitzo els principis d'UX per crear productes digitals. Això pot significar utilitzar esbossos, prototips i eines de disseny per crear els plans del que una pàgina web o una aplicació ha de semblar.
-   **Programar el front-end:** Els enginyers UX som diferents dels dissenyadors UX perquè sabem programar. Utilitzo llenguatges de programació del front-end com HTML, CSS o JavaScript per crear dissenys, funcionalitats interactives o executar altres idees de disseny.
-   **Col·laborar amb els equips d'UX i de desenvolupament:** Com a professional amb habilitats tant de disseny com de programació, sovint actuo com a pont entre els dos equips. Puc treballar amb dissenyadors, programadors del front-end o back-end i redactors per desenvolupar productes. També puc realitzar proves d'usuari, proves A/B i altres formes de recerca en col·laboració amb l'equip d'UX.
