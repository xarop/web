## Context

Breu descripció del projecte: què és, qui l'usa, quin problema resol.

## Què hi vaig fer

- Reescriptura del sistema de components amb React i CSS modules.
- Migració de SCSS a CSS modern (custom properties, container queries).
- Optimització de Core Web Vitals: LCP < 1.5s, CLS = 0, INP < 100ms.
- Implementació de sistema de disseny amb tokens unificats.

## Resultats

- **40%** de millora en Largest Contentful Paint.
- **-60%** mida del bundle de CSS.
- Score de Lighthouse: 100/100/100/100.

## Stack

React · TypeScript · CSS modules · Vite · Playwright
