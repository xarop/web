**Adamo** és un operador de telecomunicacions que necessitava una presència digital sòlida i uns fluxos de registre i contractació que funcionessin de manera impecable per als seus usuaris finals.

## Enginyeria UI amb SCSS i JavaScript

La construcció de la interfície es basava en una arquitectura CSS modular amb **SCSS/Less**, garantint coherència visual en tota la plataforma. La capa d'interacció, implementada amb **JavaScript**, cobria des de validacions de formulari fins a transicions i feedback visual en temps real.

## Integració amb APIs externes

Un dels reptes principals era la integració de **fluxos de registre complexos** amb APIs externes: verificació de cobertura, validació de dades i connexió amb sistemes de gestió de clients. Calia que l'experiència fos fluïda malgrat la complexitat del backend.

## Stack i plataforma

El projecte combinava **WordPress** com a base de continguts amb components personalitzats en **Bootstrap**, adaptats al branding i als requeriments funcionals del client.

## Tecnologies

-   HTML5
-   CSS3
-   JavaScript
-   APIs REST
-   Bootstrap
-   WordPress
